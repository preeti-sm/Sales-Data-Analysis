# 📊 Sales Data Analysis Project (Excel)

## 📌 Project Overview

This is my first data analysis project based on a sales dataset.
In this project, I explored how to clean, transform, and analyze data using Microsoft Excel to extract meaningful insights.

The goal of this project was to understand sales patterns, identify trends, and present data in a structured and visual format.

---

## 🎯 Objectives

* Clean and prepare raw sales data
* Perform data transformation for better analysis
* Analyze sales performance using Excel tools
* Create interactive and insightful dashboards

---

## 🛠️ Tools & Skills Used

* Microsoft Excel
* Data Cleaning
* Data Transformation
* Pivot Tables
* Pivot Charts
* Excel Formulas

---

## 🔄 Process Followed

### 1. Data Cleaning

* Removed duplicates and handled missing values
* Standardized column formats
* Ensured data consistency

### 2. Data Transformation

* Created new columns where required
* Converted data into analysis-friendly format

### 3. Data Analysis

* Used Pivot Tables to summarize data
* Identified trends in sales, revenue, and categories

### 4. Data Visualization

* Created Pivot Charts for better understanding
* Built a simple dashboard to present insights

---

## 📊 Key Insights

* Identified top-performing products/categories
* Observed sales trends over time
* Analyzed revenue distribution across different segments
